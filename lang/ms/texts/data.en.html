Data eksport Open Food Facts

Dump pangkalan data dan eksport dijana setiap malam.

Maklumat mengenai data berbeza untuk eksport MongoDB dan CSV boleh didapati di https://world.openfoodfacts.org/data/data-fields.txt

Dump MongoDB

Data untuk semua produk boleh didapati di dump pangkalan data MongoDB.

Link : https://world.openfoodfacts.org/data/openfoodfacts-mongodbdump.tar.gz

sha256sum: https://world.openfoodfacts.org/data/sha256sum

md5sum: https://world.openfoodfacts.org/data/md5sum

Eksport Data CSV

Data untuk semua produk, atau sebahagian daripada produk, boleh dimuat turun dalam format CSV (boleh dibaca menggunakan OpenOffice, Excel dan applikasi lain) melalui borang carian maju.

Link : https://world.openfoodfacts.org/data/en.openfoodfacts.org.products.csv

Pengekodan fail adalah Unicode UTF-8. Watak yang memisahkan bidang medan adalah <tab> (tabulasi).

Eksport Data RDF

Pangkalan data juga boleh didapati dalam format RDF. pengumuman dalam Bahasa Perancis.

Link: https://world.openfoodfacts.org/data/en.openfoodfacts.org.products.rdf

Eksperimen API JSON

API JSON juga disediakan untuk membaca data untuk produk. API ini khususnya digunakan dalam applikasi untuk iPhone dan Android.

API ini adalah sebuah eksperimen dan boleh berubah. Sila beritahu kami jika anda menggunakannya, supaya kita dapat memberitahu anda tentang perubahan yang berpotensi.

URL untuk membaca data bagi sesuatu produk: https://world.openfoodfacts.org/api/v0/product/[barcode].json

Contoh: https://world.openfoodfacts.org/api/v0/product/737628064502.json

Applikasi Android dan iPhone

Kod untuk applikasi telefon bimbit Open Food Facts boleh didapati di github. Applikasi ini membolehkan pengguna mengimbas kod bar sesuatu produk, untuk melihat maklumat sesuatu produk, dan untuk mengambil dan menghantar gambar dan data untuk produk lain yang kurang bermaklumat.

Applikasi ini dibangunkan dalam HTML dan Javascript menggunakan Phonegap (Cordova) yang membolehkan penggunaan semula kod dalam semua platform telefon bimbit.

Senarai e-mel untuk data, API dan eksport

Anda juga boleh menyertai senarai e-mel untuk membincangkan API dan eksport, beritahu kami tentang penggunaan semula anda, dan dapat dimaklumkan apabila terdapat penambahbaikan atau beruabahan kepada API. Send an empty e-mail to api-subscribe@openfoodfacts.org to subscribe.
